## 💡 Key Learnings

1. **BindingList vs List**
   - BindingList<T> required for DataGridView sorting
   - List<T> causes casting errors

2. **StatusStrip Implementation**
   - Don't cast StatusStrip as ISupportInitialize
   - Use ToolStripStatusLabel for text
   - Spring property for width

3. **Async Patterns**
   - Async/await for non-blocking UI
   - Proper error handling with try-catch
   - Loading overlays improve UX

4. **Data Binding**
   - Proper binding sources prevent errors
   - BindingList enables sorting
   - Refresh data source after updates
